苹果申请证书Certificates | AppID | Profiles等详细图文教程

Posted on 2018-06-26 | Post modified: 2018-07-10 | In iOS知识 , 苹果账号上架系列2018

进入证书管理页面

申请证书

在钥匙串中创建证书申请文件

  1. 选择钥匙串访问 –> 证书助理 –> 从证书颁发机构请求证书
  2. 填写邮箱信息，选择存储到磁盘或邮件
  3. 导出CertificateSigningRequest.certSigningRequest

选择Certificates，点右上角➕

  1. Delvelopment 开发环境

    • iOS App Development 开发环境证书
    • Apple Push Notification service SSL (Sandbox) 开发环境推送证书

  2. Production 生产环境

    • App Store and Ad Hoc 发布证书
    • Apple Push Notification service SSL (Sandbox & Production) 推送证书
    • Pass Type ID Certificate Passbook需要申请的证书
    • Website Push ID Certificate
    • WatchKit Services Certificate
    • VoIP Services Certificate VoIP服务证书，用于iOS8以后的pushkit框架
    • Apple Pay Payment Processing Certificate Apple Pay交易处理证书，该证书用来在Apple Pay交易易过程中加密交易信息
  3. 选择App Store and Ad Hoc –> continue✖2 –> 选择上一步生成的CertificateSigningRequest.certSigningRequest–> continue

App ID

选择Identifiers，点右上角➕

  1. 设置App ID 信息

    • App ID Description （App ID 的描述 英文字符）
    • App ID Suffix
      Explicit App ID (唯一ID com.xxx.xxx)
      Wildcard App ID (通用ID eg:com.xxx.*)

  2. App Services 按需勾选功能

  3. 创建成功，App ID 对应工程中 Bundle Identifier

Provisoning Profiles

选择Profiles，点右上角➕

  1. 选择Distribution 下App Store选项，如果制作开发环境的profile，选择Development内的选项
  2. 选择上一步生成的App ID
  3. 选择发布证书
  4. 设置Profile Name
  5. continue 创建成功后下载到本地，双击文件便可安装到Xcode中

Device

选择Devices，点右上角➕

  1. Register Device
    • 单个设备添加 Name / UDID
  2. Register Mutiple Devices
    • 批量添加设备 下载sample files 添加
  3. 设备UDID获取方式
    • iphone/ipad连接到电脑
    • Xcode –> Window –> Devices and Simulators
    • 设备信息中的Identifier便为UDID
